# Kapital Bank Layihəsi Test Strategiyası
## Giriş 
Bu sənəd Kapital Bank mobil tətbiqinin keyfiyyət təminatı prosesini əks etdirir.
## 2 Testin əhatə dairəsi (Scope)
**GIRIŞ MODULU:** Şifrə, OTP və Biometrik Giriş.
**KÖÇÜRMƏLƏR:** Kartdan-karta və ölkədaxili köçürmələr.
**ÖDƏNİŞLƏR:** Kommunal və QR ödənişlər.
## 3 Test Növləri
**MANUAL TESTİNG:** UI və Funksionallıq yoxlanışları.
**API TESTING:** Postman vasitəsi ilə backend yoxlanışı.
**REGRESSİON TESTİNG:** Hər yeni yenilənmədən sonra əsas funksiyaların yoxlanması.
